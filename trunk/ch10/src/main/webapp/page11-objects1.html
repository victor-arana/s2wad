Anonymous Objects

      var aDude = {
          fname: "Dave",
          lname: "Newton",
          age: Infinity // Notice no trailing comma!
      };
    
      var aDude = {
          fname: "Dave",
          lname: "Newton",
          age: Infinity,
          address: {
              street: "271828 E St",
              state: "Confused",
              zip: "69042"
          }
      };

Objects are open

We can also add properties to existing objects, like to aDude:

      aDude.baz = "Plugh";

Object properties can be functions

Lastly, object values can be functions.

      aDude.toString = function () {
          return "aDude [fname=" + this.fname + ", lname=" + this.lname + "]";
      }
